# Skill Creator V2

Skill Creator V2 es un meta-skill para crear skills de IA y grupos de skills con disciplina de produccion.

Es util cuando necesitas un skill documentado, comprobable, consciente de sus dependencias y honesto sobre la evidencia, no solo un prompt que parece terminado.

Uso: instala el paquete npm, copia el skill en tu runtime y pide al agente crear o mejorar un skill. El sistema clasifica la solicitud, carga solo las referencias necesarias, verifica herramientas y dependencias, ejecuta lint/evals cuando corresponde y genera un informe de preparacion.

Resultado esperado: un paquete estructurado con `SKILL.md`, referencias, scripts, evals, pruebas y una revision basada en evidencia.

Ejemplo: el grupo UI Intelligence paso de recopilar referencias superficiales a revisar sitios reales, capturas por scroll, metadata publica de codigo/fuentes/efectos, controles de originalidad y ciclos de mejora de los propios skills.
