# Skill Creator V2

Skill Creator V2 est un meta-skill pour construire des skills IA et des groupes de skills avec une discipline de production.

Il aide quand vous voulez un skill documente, testable, attentif aux dependances et honnete sur les preuves, au lieu d'un simple prompt qui semble termine.

Utilisation: installez le paquet npm, copiez le skill dans votre runtime, puis demandez a l'agent de creer ou d'ameliorer un skill. Il classe la demande, charge seulement les references necessaires, verifie les outils et dependances, lance les lint/evals utiles et produit un rapport de preparation.

Resultat attendu: un paquet structure avec `SKILL.md`, references, scripts, evals, tests et review fondee sur des preuves.

Exemple: le groupe UI Intelligence a transforme une collecte superficielle de references en workflow avec sites ouverts, captures d'ecran, metadata publique de code/polices/effets, controles d'originalite et boucles de retour vers les skills.
