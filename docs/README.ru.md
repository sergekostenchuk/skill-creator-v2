# Skill Creator V2

Skill Creator V2 — это мета-скилл для создания других AI-скиллов и групп скиллов с production-дисциплиной.

Он полезен, когда нужен не просто красивый prompt, а документированный, проверяемый, безопасный по зависимостям и честный по доказательствам скилл.

Как пользоваться: установите npm-пакет, скопируйте скилл в нужный runtime и попросите агента создать или улучшить скилл. Skill Creator V2 классифицирует задачу, подгружает нужные reference-файлы, проверяет tools/dependencies, запускает lint/evals где это применимо и формирует отчет готовности.

Ожидаемый результат: пакет со `SKILL.md`, справочниками, скриптами, evals, тестами и evidence-backed review.

Пример: группа UI Intelligence благодаря этому подходу ушла от поверхностного сбора референсов к live-site evidence, screenshot review, metadata по публичному коду/шрифтам/эффектам, originality checks и циклу улучшения самих скиллов.
