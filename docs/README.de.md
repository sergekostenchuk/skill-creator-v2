# Skill Creator V2

Skill Creator V2 ist ein Meta-Skill zum Erstellen von KI-Skills und Skill-Gruppen mit Produktionsdisziplin.

Er ist hilfreich, wenn ein Skill dokumentiert, testbar, abhaengigkeitsbewusst und ehrlich ueber Evidenz sein soll, statt nur wie ein fertiger Prompt auszusehen.

Nutzung: Installiere das npm-Paket, kopiere den Skill in die Zielumgebung und bitte den Agenten, einen Skill zu erstellen oder zu verbessern. Der Skill klassifiziert die Anfrage, laedt nur die noetigen Referenzen, prueft Tools und Abhaengigkeiten, fuehrt passende Lint/Eval-Schritte aus und erzeugt einen Readiness-Report.

Erwartetes Ergebnis: ein strukturiertes Paket mit `SKILL.md`, Referenzen, Skripten, Evals, Tests und evidenzbasierter Review.

Beispiel: Die UI-Intelligence-Gruppe entwickelte sich von oberflaechlicher Referenzsammlung zu Live-Site-Evidenz, Screenshot-Review, oeffentlicher Code-/Font-/Effekt-Metadata, Originalitaetspruefung und Feedback-Schleifen zur Verbesserung der Skills.
