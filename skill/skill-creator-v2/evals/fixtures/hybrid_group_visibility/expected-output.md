# Expected Output

The skill creator must choose `group_layout: hybrid_group`.

Expected visibility:

- one visible installable orchestrator skill;
- tightly coupled source routing, pattern inventory, and originality review roles as `internal_worker` entries using `WORKER.md`;
- reusable technology/effect inspection as `reusable_satellite_skill` because it has a standalone trigger and output;
- source registry, artifact contract, and originality policy as `shared_module`;
- minor copy or formatting steps as `merge_into_parent`.

Forbidden:

- do not create standalone `SKILL.md` folders for every internal worker;
- do not place `SKILL.md` under `group/workers/**`;
- do not turn shared policy/reference files into skills.
